## Usage
Search for Chocolatey packages using "**choco** [search term]".

* <kbd>Return</kbd> - Opens an admin PowerShell console and installs package
* <kbd>Ctrl+C</kbd> - Copy the install command eg. `choco install Firefox -yv`
